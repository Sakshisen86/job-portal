# Job Portal

A full-stack MERN Job Portal application where job seekers can search and apply for jobs, while recruiters can post and manage job listings.

# Job Seeker
- User Registration & Login
- Secure JWT Authentication
- Browse Available Jobs
- Search & Filter Jobs
- Apply for Jobs
  
# Recruiter/Admin
- Recruiter Registration & Login
- Create Company
- Update Company Details
- Post New Jobs
- Manage Posted Jobs
